# Zepto Sales Excel Dashboard

## Project Overview
This project analyzes Zepto sales data using Microsoft Excel and creates an interactive dashboard to visualize business insights such as sales trends, order status, customer demographics, and payment methods.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers
- Data Cleaning

## Dashboard Features
- Orders vs Sales analysis by month
- Order status distribution (Delivered, Cancelled, Returned)
- Customer Age vs Gender analysis
- Payment method analysis
- Top 5 performing states

## Dataset Information
The dataset includes the following fields:
- Order ID
- Order Date
- Month
- Customer ID
- State
- Category
- Product Name
- Amount
- Gender
- Age
- Age Group
- Order Status
- Payment Method

## Files in Repository
- Raw_Dataset.xlsx
- Cleaned_Dataset.xlsx
- Zepto sales Dashboard.png

## Key Insights
- Most orders are successfully delivered.
- Credit Card is one of the most used payment methods.
- Pune and Ahmedabad generate high sales.
- Adults are the major customer segment.

## Project Purpose
The goal of this project is to demonstrate data analysis and dashboard creation skills using Excel for business decision-making.
